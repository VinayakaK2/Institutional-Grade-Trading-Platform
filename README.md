# Institutional Swing Trading Platform

## Vision
To build a production-grade, autonomous algorithmic trading platform designed for institutional-level stability, predictability, and performance. This platform operates on deterministic trading rules augmented by an intelligence layer for analysis and continuous improvement.

## Architecture Overview
The platform follows a Modular Monolith architecture, leveraging Clean Architecture principles to separate core business logic from infrastructure and external interfaces.
