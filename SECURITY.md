# Security Policy

## Supported Versions

Only the latest release (or frozen stable branches) receives security updates.

## Reporting a Vulnerability

Please do not report security vulnerabilities through public GitHub issues.

If you believe you have found a security vulnerability, please report it via our dedicated security contact or internal security tracking system. All reports will be evaluated and acknowledged promptly. We practice responsible disclosure and ask that you wait until a fix is deployed before publicly disclosing any findings.
